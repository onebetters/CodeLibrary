# LeetCode

分为简单和中等，因为大厂的算法一般也就是中等难度，很多算法都是在中等(算法岗除外)

每个题都是通过了测试，但是代码的风格质量可能不是最优的

总共就一个项目，项目分类按照包分类，分为

- 简单
  - 数组
  - 链表
  - 哈希
  - 二叉树
  - 等.. 
- 中等

因为题目不可能是孤立的（比如一个题可能同时要用到数组和hash），所以只是大致分了下，我用什么标签搜索的就放进了哪个包

每一个类就是一个题目，每个类中的注释都会有 

- 题目描述
- 示例
- 思路
- 代码
